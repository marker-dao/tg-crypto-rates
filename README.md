## Getting Started

```bash
# clone the repository
git clone git@github.com:username/tg-crypto-rates.git
cd tg-crypto-rates

# install dependencies
pnpm install

# start the dev server
pnpm dev

# build for production
pnpm build

# preview the production build
pnpm preview
```
